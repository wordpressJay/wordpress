== Description ==

Fleming is a mobile friendly (responsive), versatile and flexible WordPress theme with a solid, classic two column layout. This theme is perfect for schools, universities, kindergartens, churches, organizations, non-profits, corporate and other websites that need an intuitive and professional look.


== Installation ==

1. In your admin panel, go to Appearance > Themes and click the Add New button.
2. Click Upload and Choose File, then select the theme's .zip file. Click Install Now.
3. Click Activate to use your new theme right away.

= How to setup the front page like the demo site? =

The demo site URL: https://demo.academiathemes.com/fleming-lite/

When you first activate Fleming, your homepage will display posts in a traditional blog format. If you would like to add the additional featured sections from the demo, you can do that by going to Appearance > Customize > Theme Options.

= Footer Area =

The footer area contains 4 widgetized columns of equal width.

== Quick Specs (all measurements in pixels) ==

	1. The main column width is 600.
	2. In full-width layout the content width is 1200.
	3. A widget in the sidebar is 250.
	4. The featured image on the front page and on pages works best with images at least 1200 wide.
	5. Featured Images for posts should be at least 580 wide.
	6. The footer widget columns are of equal width: 260.
